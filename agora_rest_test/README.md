agora_rest_test
======


Prerequisites
-------------

To get ready to build, you will have to have the following:

* [Go 1.8.0](http://golang.org/) or later
* [GPM](https://github.com/pote/gpm)

Building
--------

To build, you can then run:

    $ make


To clean
	$ make clean

To start over
    $ make over

To run HTTP
    $ make run

To run HTTPS
    $ make run_secure
